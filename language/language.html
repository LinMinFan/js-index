js語法

變數/常數

let 宣告變數

const 宣告常數

全域變數 不在函數內的變數

區域變數 函數內宣告只在這一個函數區域內可用

取值

getElementById 抓單個id

getElementsByClassName 全部的class

querySelector 抓單個id 要#id

querySelectorAll 抓class 要.class

資料類別

string 字串

number 數字

boolean 布林

object 物件

分辨方式:log中輸入typeof

字串運算式

利用 + 號組合

"a!"+"b!" // 輸出結果a!b!

跳脫字元

\b 倒退

\f 換頁

\n 換行

\' 單引號

\" 雙引號

\\ 反斜線
