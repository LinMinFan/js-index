js陣列語法

  • 創建陣列

    • let array_name = [item1, item2, ...];

  • 呼叫陣列

    • array_name[0,1,2...];

  • 長度屬性

    • array_name.length;

  • Array.forEach()功能

    • array.forEach((value,key) => { });

  • 從最後添加陣列value

    • array.push("value");

  • 陣列刪除最後一個value

    • array..pop();

  • 識別陣列value

    • typeof array => object;

  • 陣列轉換為字串

    • array.toString();

  • 陣列合併

    • newarray = arrayA.concat(arrayB);A陣列後加上B陣列

  • 陣列拼接

    • splice(2, 0, "value1", "value2");

    第一個參數(2)定義了應該添加/刪除新元素的位置。

    第二個參數(0)定義應該刪除多少元素。

    array.splice(2,2);

    回傳從(2)開始刪除的(2)項value

  • 按字母順序對陣列進行排序

    • array.sort();

  • 反轉陣列

    • array.reverse();

  • 陣列映射

    • array.map(myFunction(value, index, array){return value + 條件(運算)});

    通過對每個數組元素執行一個函數來創建一個新數組。

    function內需要3個參數,若只使用value參數時,可以省略index和array參數

  • 陣列過濾器

    • array.filter(myFunction(value, index, array){return value + 條件(比較值)});

    function內需要3個參數,若只使用value參數時,可以省略index和array參數

  • 陣列數字的總和

    • array.reduce(myFunction(total,value, index, array){return total + value});

    function內需要4個參數,若只使用value參數時,可以省略index和array參數

    可以接受一個初始值array.reduce(myFunction, 初始值); => 初始值 + sum

  • 陣列中搜索元素值並返回其位置

    • array.indexOf;

    返回索引值(0,1,2...)如果未找到該項目,則返回(-1)多次出現,則返回第一次出現的位置。

  • 陣列包含

    • array.includes;

    檢查一個元素是否存在於數組中,包括 NaN,與indexOf不同回傳布林值
