物件基本概念

物件導向

定義物件類別=>定義物件屬性=>定義物件方法

Object有如PHP中的關聯陣列

  • 創建方式

    • const person = {firstName:"John", lastName:"Doe", age:50,eyeColor:"blue"};

    const person = {
    firstName: "John",
    lastName: "Doe",
    age: 50,
    eyeColor: "blue"
    };

    const person = {};
    person.firstName = "John";
    person.lastName = "Doe";
    person.age = 50;
    person.eyeColor = "blue";

    const person = new Object();
    person.firstName = "John";
    person.lastName = "Doe";
    person.age = 50;
    person.eyeColor = "blue";

物件屬性

  • 呼叫屬性語法

    • objectName.property // person.age

    objectName["property"] // person["age"]

    objectName[expression] // x = "age"; person[x]

for...in函式 / for...of函式

  • 語法

    • for (const key in object) {
    console.log(key);
    } //提出索引/關聯索引

    for (const iterator of object) {

    } //提出屬性(僅陣列適用)物件不適用此函式(物件可用Object.values())

增加 / 刪除 物件屬性

  • 增加新屬性

    • person.newproperty = "English";

  • 刪除屬性

    • delete person.property;

二維物件

  • 範例

    • myObj = {
    name:"John",
    age:30,
    cars: {
    car1:"Ford",
    car2:"BMW",
    car3:"Fiat"
    }
    }

  • 呼叫方式

    • myObj.cars["car2"];

    myObj["cars"]["car2"];

物件中可以是陣列,陣列中也可以是物件

  • 範例

    • const myObj = {
    name: "John",
    age: 30,
    cars: [
    {name:"Ford", models:["Fiesta", "Focus", "Mustang"]},
    {name:"BMW", models:["320", "X3", "X5"]},
    {name:"Fiat", models:["500", "Panda"]}
    ]
    }

Object.values()

  • 範例

    • const person = {
    name: "John",
    age: 30,
    city: "New York"
    };

    const myArray = Object.values(person);
    document.getElementById("demo").innerHTML = myArray;

this指向變數

this 值在所有函式以外的全域執行環境下，會被當作全域物件

console.log(this.document === document); // true

// 在網路瀏覽器中，window 物件也是全域物件。
console.log(this === window); // true

this.a = 37;
console.log(window.a); // 37

this.b = "MDN";
console.log(window.b) // "MDN"
console.log(b) // "MDN"

在函式內的 this 值取決於該函式如何被呼叫

let a = 15;
function myfun(a){
let x = a;
let y = this.a;
console.log(x,y);
}
myfun(100); //x = 100 , y =15;
