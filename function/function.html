函式定義/呼叫

  • 定義

函數是用function關鍵字定義的，後跟一個name，然後是括號()。
函數名稱可以包含字母、數字、下劃線和美元符號（與變量相同的規則）。
括號中可能包含以逗號分隔的參數名稱：(
parameter1 , parameter2, ... )
函數要執行的代碼放在大括號內：{}

function 函式名稱()
{
程式敘述;
return 回傳值; //可有可無
}

當碰到一個return語句時，函數將停止執行。

函數也可以使用表達式來定義。

const x = function (a, b) {return a * b};
變數x可當作function_name

  • 呼叫函式

function_name();

  • 自動啟動函式不須呼叫

(function () {
let x = "Hello!!"; // I will invoke myself
})();

  • 箭頭函式

箭頭函式為函式表達式的簡短語法。
不需要function關鍵字、return關鍵字和大括號。

let a = function (x,y){
return x * y; }

縮減後為
let a = (x, y) => x * y;

箭頭函式沒有自己的this. 不適合定義對象方法。
箭頭函式沒有被提升。它們必須在使用前定義。
const比let，因為函式表達式始終是常量值。

  • 函式參數

函式定義不指定參數的數據類型。
函式不對傳遞的參數執行類型檢查。
函式不檢查接收到的參數數量。
函式時缺少參數，有時這是可以接受的，但有時最好為參數分配一個默認值。

function myFunction(x, y = 2) {
程式敘述;
}
